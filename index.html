📲 Install As App
Follow these steps on any device. No server app required — this hub lives on the internet.
1
Open Chrome on your Android
Must be Chrome — not Firefox or Samsung Browser.
2
Go to the hub URL
https://squishface.github.io/rehub/
3
Add to Home Screen
Tap the three-dot menu ⋮ in Chrome → "Add to Home screen" → name it Kevin's Real Estate Hub → tap Add.
✓ Opens full-screen like a native app. No server needed.
4
Adding new agents in future
Go to github.com/Squishface/rehub on any browser → drag and drop your new .html file → tap Commit. The hub updates automatically.
💡 You can delete files the same way — just select the file and delete it on GitHub.
＋ Add Agent
Upload the .html file to GitHub first, then register it here as a card.
📂 To upload: go to github.com/Squishface/rehub → drag your file in → tap Commit Changes.
